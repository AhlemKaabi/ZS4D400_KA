# ABAP package - SAP Learning

Working on a “Hello World” ABAP Cloud project using a trial instance of the ABAP environment on SAP BTP.

The ABAP environment instance is created from the SAP BTP Cockpit in a **trial account**. You can access the cockpit and create your account [HERE](https://account.hanatrial.ondemand.com/).

* The development setup starts by installing ABAP Development Tools (ADT) in the **Eclipse** IDE.

* After installing ADT, You create an **ABAP Cloud Project** in Eclipse to connect to the ABAP service instance running in SAP BTP (you need to have the Project URL).

* You will be asked to logon to the service instance, once authenticated, the ABAP system appears in the Eclipse Project Explorer.

After the connection is established:
- Create an ABAP package in the system to store development objects.
- Inside the package, create an ABAP class that contains the code for the application.

In this project, the class implements a simple Hello World console application.

*Project was pushed to GitHub using abapGit*
